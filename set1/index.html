Coerción

Es la forma en que podemos cambiar un tipo de valor a otro tipo de valor, y existen dos tipos de coercion:

  • Coerción implicita
  • Coerción Explicita

Coerción Implicita

Es cuando el lenguaje nos ayuda y cambia de untipo de valor otro tipo de valor 4 + "7"; // "47" aqui el lenguaje asume que se concatena el 4 a una cadena de texto 4 * "7"; // 28 en esta si asume que es una multiplicacion y cambia el tipo del valor "7" de cadena a numero

Coerción Explicita

Es la forma en que nosotros obligamos a que un valor de un tipos cambio a otro valor de otro tipo
Ejemplo obligar al cambio de tipos var a = 20; // queremos que el numero 20 se convierta en un string sin //necesidad de hacer esto //var b = a + ""; var c = String (a); typoof c --> String // el metodo String es una coerción explicita
